  2. Paragraph(<p>)

    • Groups sentences & text blocks together.
    • Browsers automatically add spacing around <p>.
    • Use multiple <p> tags instead of line breaks for new paragraphs.

    Example:

                <p>This is the first paragraph. It contains several sentences about a topic.</p>

  3. Horizontal Rule (<hr>)

    • Defines a thematic break in the page.
