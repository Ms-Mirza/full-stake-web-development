📘 Notes on HTML Nested Lists & Tables

✨ Part 1: HTML Nested Lists

🔹 Tags Used in Nested Lists

  • <ul> →Defines an Unordered list (bullets).
  • <ol> → Defines an Ordered list (numbers/letters).
  • <li> → Defines a List item (inside <ul> or <ol>).
  • Nested list = A list inside another <li>.

🔹 Example 1: Unordered List inside Unordered List

    <ul>
        <li>Fruits
            <ul>
                <li>Apple</li>
                <li>Banana</li>
                <li>Orange</li>
            </ul>
        </li>
        <li>Vegetables
            <ul>
                <li>Carrot</li>
                <li>Potato</li>
                <li>Tomato</li>
            </ul>
        </li>
    </ul>

🔹 Example 2: Ordered List inside Unordered List

        <ul>
            <li>Steps to Make Tea
                <ol>
                    <li>Boil Water</li>
                    <li>Add Tea Leaves</li>
                    <li>Add Sugar & Milk</li>
                    <li>Serve Hot</li>
                </ol>
            </li>
        </ul>

✨ Part 2: HTML Tables

🔹 Tags Used in Tables

  • <table> → Creates a table.
  • <tr> → Defines a table row.
  • <th> → Defines a table header cell (bold & centered by default).
  • <td> → Defines a table data cell.
  • <td> → Defines a table data cell.

🔹 Example 1: Simple Table

            <table border="1">
            <tr>
                <th>Roll No</th>
                <th>Name</th>
                <th>Marks</th>
            </tr>
            <tr>
                <td>1</td>
                <td>Rahim</td>
                <td>85</td>
            </tr>
            <tr>
                <td>2</td>
                <td>Karim</td>
                <td>90</td>
            </tr>
            </table>

🔹 Example 2: Nested Table

                <table border="1">
                    <tr>
                        <th>Name</th>
                        <th>Details</th>
                    </tr>
                    <tr>
                        <td>Rahim</td>
                        <td>
                            <table border="1">
                                <tr>
                                    <td>Subject</td>
                                    <td>Marks</td>
                                </tr>
                                <tr>
                                    <td>Math</td>
                                    <td>90</td>
                                </tr>
                                <tr>
                                    <td>Science</td>
                                    <td>85</td>
                                </tr>
                            </table>
                        </td>
                    </tr>
                </table>

📝 Summary

  • Use <ul>, <ol>, and <li> for nested lists.
  • Use <table>, <tr>, <th>, and <td> for tables.
  • Both lists and tables can be nested to organize complex information.
